### Launchers

- **super + shift + alt + l** : lock screen (slock)  
- **super + Return** : open terminal (alacritty)  
- **super + p** : open rofi launcher  
- **super + Print** : open flameshot gui  

### System

- **super + Escape** : reload sxhkd  
- **super + ctrl + shift + r** : reload bspwm and sxhkd  

### Desktop Management

- **super + [1-9]** → focus on desktop [1-9]  
- **super + shift + [1-9]** → move focused window to desktop [1-9]  

### Bspwm Control

- **super + ctrl + shift + Escape** : quit bspwm  
- **super + shift + q** : close window  
- **super + shift + ctrl + q** : kill window  
- **super + m** : toggle between tiled and monocle layout  
- **super + y** : send marked node to newest preselected node  
- **super + g** : swap current node with biggest window  
- **ctrl + alt + h** : hide/unhide window  

### State / Flags

- **super + shift + space** : toggle floating  
- **super + shift + f** : toggle fullscreen  
- **super + shift + t** : toggle tiled  
- **super + shift + m** : toggle marked  
- **super + shift + x** : toggle locked  
- **super + shift + 0** : toggle sticky  
- **super + shift + z** : toggle private  

### Focus / Swap

- **super + p** : focus parent  
- **super + b** : focus brother  
- **super + comma** : focus first  
- **super + period** : focus second  
- **super + j** : focus next window  
- **super + k** : focus previous window  
- **super + shift + j** : swap with next window  
- **super + shift + k** : swap with previous window  
- **super + s** : focus next monitor  
- **super + shift + s** : move node to next monitor and focus it  
- **super + grave** : focus last node  
- **super + Tab** : focus last desktop  

### Preselection

- **super + alt + h/j/k/l** : preselect direction (west/south/north/east)  
- **super + alt + [1-9]** : preselect ratio 0.[1-9]  
- **super + alt + space** : cancel node preselection  
- **super + ctrl + shift + space** : cancel all preselection on current desktop  

### Move / Resize

- **super + ctrl + h/j/k/l** : expand window (left/bottom/top/right)  
- **super + ctrl + shift + h/j/k/l** : contract window (right/top/bottom/left)  

### Utilities

- **alt + F1** : open key help  
- **alt + space** : open theme selector  
- **alt + e** : open rice editor  
- **super + alt + t** : open terminal selector  
- **button3 on desktop** : open jgmenu  
- **super + alt + o** : open scratchpad terminal  
- **super + alt + h** : hide bar  
- **super + alt + u** : show bar  
- **super + alt + c** : open clipboard manager  
- **super + alt + p** : open power menu  
- **super + alt + k** : open keyboard layout applet  

### Multimedia

- **XF86AudioRaiseVolume** : increase volume  
- **XF86AudioLowerVolume** : decrease volume  
- **XF86AudioMute** : toggle mute  
- **XF86AudioNext** : next track  
- **XF86AudioPrev** : previous track  
- **XF86AudioPlay** : play/pause  
- **XF86AudioStop** : stop  

### Window Search

- **super + shift + w** : search active windows
